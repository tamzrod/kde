# Meeting Notes

This directory contains meeting notes from research discussions.

## Meeting File Naming Convention

```
YYYY-MM-DD-[brief-description].md
```

Example:
- `2024-01-15-kickoff-research.md`
- `2024-02-20-tier1-knowledge-review.md`

## Meeting Template

```markdown
# Meeting: [Title]

## Date
[Date and time]

## Attendees
- [Name 1]
- [Name 2]

## Agenda
1. [Topic 1]
2. [Topic 2]

## Discussion Summary
[Summary of key discussion points]

## Decisions Made
- [Decision 1]
- [Decision 2]

## Action Items
- [ ] [Action 1] - Owner: [Name] - Due: [Date]
- [ ] [Action 2] - Owner: [Name] - Due: [Date]

## Next Meeting
[Date and topic for next meeting]
```
