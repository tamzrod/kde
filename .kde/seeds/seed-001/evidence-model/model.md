## Overview

Evidence is the documented facts that support knowledge claims. This model defines what counts as evidence within KDE.

---

## What is Evidence?

Evidence is **documented fact** that can be verified and traced to a source.

### Evidence Properties

| Property | Description |
|----------|-------------|
| **Documented** | Written down and preserved |
| **Factual** | Observable or verifiable |
| **Sourced** | Traced to an origin |
| **Verifiable** | Can be checked for accuracy |
| **Linkable** | Connected to experiments or claims |

---

## Evidence vs Other Concepts

| Concept | Definition | Evidence Difference |
|---------|------------|---------------------|
| **Evidence** | Documented facts | Observable reality |
| **Inference** | Conclusions from evidence | Interpretation |
| **Hypothesis** | Speculation | Not yet supported |
| **Knowledge** | Validated evidence | Approved understanding |

## Evidence Types

### 1. Observational Evidence

Direct observations from experiments:

```
Experiment observation ──documents──► Evidence
```

| Property | Description |
|----------|-------------|
| Source | Laboratory experiments |
| Format | Observation records |
| Verification | Reproducibility checks |

### 2. Research Evidence

Findings from literature review:

```
Research review ──documents──► Evidence
```

| Property | Description |
|----------|-------------|
| Source | Academic papers, reports |
| Format | Citation + summary |
| Verification | Source credibility |

### 3. Validation Evidence

Results from knowledge testing:

```
Validation test ──produces──► Evidence
```

| Property | Description |
|----------|-------------|
| Source | Laboratory experiments |
| Format | Test results |
| Verification | Statistical analysis |

## Evidence Quality

### Quality Criteria

| Criterion | Description | Weight |
|-----------|-------------|--------|
| **Verifiability** | Can be checked | High |
| **Reproducibility** | Can be repeated | High |
| **Source Credibility** | Reliable origin | Medium |
| **Relevance** | Applies to claim | High |
| **Completeness** | Full picture | Medium |

### Quality Assessment

Evidence is assessed on:
- Source reliability
- Method soundness
- Result consistency
- Reproducibility

---

## Evidence Collection

### Collection Process

```
1. Identify claim needing support
         │
         ▼
2. Determine evidence requirements
         │
         ▼
3. Collect from appropriate source
         │
         ▼
4. Document evidence with citation
         │
         ▼
5. Verify evidence integrity
         │
         ▼
6. Link to experiment or claim
```

### Documentation Requirements

Every evidence item must include:
- Source citation
- Collection date
- Verification status
- Link to relevant experiment
- Integrity hash

## Evidence Hierarchy

| Level | Type | Weight |
|-------|------|--------|
| **Level 1** | Direct observation | Highest |
| **Level 2** | Reproduced experiment | High |
| **Level 3** | Peer-reviewed source | Medium |
| **Level 4** | Expert opinion | Lower |
| **Level 5** | Unverified claim | Lowest |

---

## Evidence Standards

### Required Standards

| Standard | Description |
|----------|-------------|
| **Traceability** | Every claim traces to evidence |
| **Verification** | Evidence can be verified |
| **Linkage** | Evidence connected to experiments |
| **Preservation** | Evidence is permanently stored |
| **Integrity** | Evidence unchanged after collection |

### Prohibited

| Prohibition | Reason |
|-------------|--------|
| Unverified claims | Cannot be trusted |
| Uncited assertions | No source |
| Modified evidence | Integrity violation |
| Unlinked evidence | Orphaned data |

---

## Evidence in Knowledge

### Knowledge Requires Evidence

Every knowledge claim must:
1. Cite supporting evidence
2. Document evidence quality
3. Link to experiments (if applicable)
4. Maintain evidence chain

### Evidence Weight

Evidence contributes to:
- Knowledge confidence
- Validation decisions
- Recommendation strength
