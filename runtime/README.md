# KDE Runtime: Knowledge-on-Demand

This directory contains the Knowledge-on-Demand Runtime implementation.

## Components

| Component | Description |
|-----------|-------------|
| `catalog.json` | Knowledge artifact index |
| `retrieval.py` | Retrieval engine |
| `sop005.py` | SOP-005 retrieval policy execution |
| `instrumentation.py` | Runtime logging and metrics |
| `attribution.py` | Decision origin tracking |
| `runtime.py` | Main runtime integration |

## Usage

```bash
# Initialize runtime for investigation
python runtime.py init --investigation INV-020

# Retrieve knowledge for context
python runtime.py retrieve --investigation INV-020 --domain architecture

# Log retrieval event
python runtime.py log --event retrieval --investigation INV-020

# Generate metrics report
python runtime.py metrics --investigation INV-020
```
