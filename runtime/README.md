# KDE Runtime: Knowledge-on-Demand + Skills + Orchestration

This directory contains the complete KDE Runtime implementation.

## Architecture

```
User Request
    │
    ▼
RuntimeOrchestrator.orchestrate()
    │
    ▼
TaskClassifier.classify()
    │   (13 task types, pattern matching)
    ▼
SkillSelector.select()
    │   (Task → Skills mapping)
    ▼
KnowledgeOnDemandRuntime.initialize()
    │   (SOP-005, knowledge retrieval)
    ▼
Build Execution Context
    │
    ▼
Execute Engine (unchanged)
    │
    ▼
Return Result
```

## Usage

### Complete Orchestration

```python
from runtime.orchestrator import RuntimeOrchestrator

orchestrator = RuntimeOrchestrator()
result = orchestrator.orchestrate(
    "Investigate whether KDE can improve frontend quality"
)
print(f"Task: {result.classification.task_type.value}")
print(f"Skills: {result.skill_selection.skills}")
print(f"Knowledge: {result.knowledge_retrieved_count} artifacts")
```

### Individual Components

```python
# Knowledge-on-Demand
from runtime.runtime import KnowledgeOnDemandRuntime
runtime = KnowledgeOnDemandRuntime('INV-023')
context = runtime.initialize(title="...", keywords=[...])

# Dynamic Skills
from runtime.skills import SkillLoader
loader = SkillLoader()
skills = loader.select_skills_for_task(['continuation'])
```
