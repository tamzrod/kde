## Version History

### v0.1.0 (2026-07-20) — Initial Release

**Status**: Active

#### Changes

- Initial documented engine
- First official KDE Engine established
- Baseline methodology captured from project inception through LAB-011

#### What Changed

This is the first engine release. No previous engine exists.

#### Previous State

No previous engine. This is the founding engine of the KDE Engine Framework.

#### Rationale

Prior to this engine, KDE methodology existed only as practice without formal versioning. This engine:

1. Documents existing methodology in a formal structure
2. Establishes the Engine Framework for future evolution
3. Provides provenance for all existing experiments
4. Creates foundation for methodology improvements

#### Breaking Changes

None. This engine defines the baseline.

#### Migration

No migration required. All existing experiments (LAB-001 through LAB-011) have been migrated to reference this engine.

## Version Conventions

### Version Number Format

Format: MAJOR.MINOR.PATCH

| Component | Description | When It Changes |
|-----------|-------------|-----------------|
| **MAJOR** | Breaking changes | Fundamental methodology changes |
| **MINOR** | Additive improvements | New requirements or processes |
| **PATCH** | Clarifications | Non-substantive improvements |

### Codename Conventions

| Codename | Meaning |
|----------|---------|
| **Alpha** | Initial development stage |
| **Beta** | Stable, actively developed |
| **Release** | Production-ready |
| **Legacy** | Historical, no longer developed |
