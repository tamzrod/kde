# Knowledge

The Knowledge folder is the canonical location for validated concepts discovered through research.

## Purpose

Knowledge contains definitions and concepts that have been validated through the research process. Unlike a glossary (which is a presentation of terminology), Knowledge is the primary artifact of validated understanding.

## Format

```markdown
# [Validated Concept]

## Definition
[Validated definition]

## Evidence
- [Evidence supporting this definition]

## Dependencies
- [Related concepts]
- [Questions that led to this definition]

## Validated
- **Date**: [Validation date]
- **Process**: [How it was validated]

## Notes
[Any nuances or edge cases]
```

## Status

Knowledge is populated as research progresses through the question tiers.

## Validation Criteria

A concept is promoted to Knowledge when:
1. Supported by multiple sources of evidence
2. Has survived counter-example testing
3. Has been reviewed by peers
4. Has no unresolved objections

## Relationship to Other Folders

- **Research**: Source of questions and investigations
- **Laboratory**: Where ideas are experimented with
- **Deployment**: Where validated concepts become artifacts
