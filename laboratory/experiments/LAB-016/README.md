## Research Question

## Task

Each run shall independently create a completely new science-fiction universe.

Each run must generate:
- **World**: The setting and environment
- **Timeline**: Key events and dates
- **Geography**: Locations and regions
- **Organizations**: Factions and groups
- **Technology**: Scientific principles and devices
- **Vehicles**: Ships, vehicles, spacecraft
- **Characters**: Named individuals
- **Canonical Facts**: Verified knowledge
- **Evidence**: Sources and references
- **Ambiguities**: Unresolved questions
- **Contradictions**: Internal conflicts

**No information may be copied from previous Laboratory experiments.**

Every conclusion MUST reference evidence and run IDs.

Unsupported conclusions are PROHIBITED.
