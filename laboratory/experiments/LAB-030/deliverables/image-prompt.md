## Primary Image Prompt

Use this prompt to generate a professional LinkedIn header image:

```
Professional LinkedIn post header image for "Knowledge Discovery Engine" (KDE) research framework introduction.

Design specifications:
- Dimensions: 1200x627 pixels (optimal for LinkedIn feed)
- Color palette: Deep navy blue (#1a237e) as primary, teal accent (#00897b), white text on dark sections
- Typography: Clean sans-serif, bold headline text

Composition:
- Left side: Abstract representation of a "scientific loop" with arrows forming a circular process - showing evidence → hypothesis → experiment → validation cycle
- Right side: Three ascending bars or nodes representing "evidence accumulation" with subtle glow effects
- Center-bottom: Small text overlay area (keep clean, minimal)

Visual elements:
- Circuit-board-inspired connecting lines in subtle gray
- Small circular nodes representing "knowledge checkpoints"
- Gradient fade on edges
- Professional matte finish, no reflections

Text elements:
- Small watermark "KDE" in bottom-right corner
- No main text overlay (text will be in LinkedIn post)

Style: Technical illustration meets modern data visualization, reminiscent of scientific paper figures but more dynamic. Think IEEE visualization meets Apple keynote aesthetics.

Avoid: Photorealistic images, people/faces, cartoon characters, excessive gradients, busy patterns
```

1. **Dimensions**: LinkedIn recommends 1200×627 pixels for optimal display
2. **Format**: PNG for best quality, JPG acceptable
3. **Text**: Keep image text-free; add text in LinkedIn post
4. **Testing**: Generate multiple variants and test engagement
