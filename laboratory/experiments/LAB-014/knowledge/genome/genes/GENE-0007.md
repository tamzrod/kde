## Canonical Statement

Reproducibility definition varies by domain type. Formal domains define reproducibility as consistent results; interpretive domains define reproducibility as consistent methodology.

---

## Description

The definition of "reproducibility" differs fundamentally between domain types:

**Formal Domains (Engineering, Science, Math)**:
- Reproducibility = consistent results
- Different practitioners should reach same conclusions
- Example: RFC 7231 specifies GET behavior; all implementations agree

**Interpretive Domains (Creative, Legal)**:
- Reproducibility = consistent methodology
- Different practitioners may make different valid choices
- Example: Different creative choices valid for same world
