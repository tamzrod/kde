## HYPOTHESIS TESTING PROTOCOL

Each run must:

1. **Reference World artifact** - Extract canonical facts from WORLD.md
2. **Assign OBS IDs** - Each observation gets unique ID
3. **Reference EV IDs** - Each observation links to evidence
4. **Detect inconsistencies** - Compare facts against each other
5. **Assess severity** - CRITICAL / HIGH / MEDIUM / LOW
6. **Provide confidence** - 0-100% with reasoning
