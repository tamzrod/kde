## ANALYSIS PROCEDURE

For EACH expression:

1. **Entity Identification** - Variables, constants, operators, functions
2. **Structural Relationships** - Inputs, outputs, dependencies, hierarchy
3. **Causal Reasoning** - Why each component exists

5. **Confidence Assignment** - With evidence and alternatives
