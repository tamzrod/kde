# Evidence Repository

This directory stores evidence collected during research.

## Organization

Evidence is organized by:
- Source type (academic, industry, philosophical, etc.)
- Related question

## Naming Convention

Files should be named:
```
[source-type]-[brief-description]-[date].[extension]
```

Examples:
- `academic-plato-knowledge-dialogues-380bc.md`
- `philosophical-russell-analysis-knowledge-1912.md`
- `scientific-popper-falsification-1934.md`

## Evidence Format

Each piece of evidence should include:

```markdown
# Evidence: [Brief Title]

## Source
- **Author**: [Name]
- **Title**: [Work title]
- **Year**: [Year]
- **Type**: [Academic paper, Book, etc.]

## Summary
[1-2 paragraph summary of the relevant content]

## Key Points
- [Key point 1]
- [Key point 2]

## Relevance to KDE Research
[Why this evidence matters for our questions]

## Direct Quotes
> "[Relevant quote from the source]"

## Notes
[Any additional observations or questions]
```

## Evidence Categories

| Category | Description |
|----------|-------------|
| `academic` | Peer-reviewed academic sources |
| `philosophical` | Philosophical texts and analyses |
| `scientific` | Scientific methodology sources |
| `industry` | Industry practices and case studies |
| `historical` | Historical examples and cases |
